## 1. 系统登录

### 1.1 超级管理员登录

访问 `https://your-domain.com` 

- 用户名：`admin`
- 密码：`admin123`（首次登录后请立即修改）

### 1.2 企业管理员登录

访问 `https://your-domain.com/admin`

- 账号：由企业超级管理员分配
- 密码：初始密码由超级管理员设置

### 1.3 员工登录

**方式一：账号密码登录**
- 打开小程序/H5/APP
- 输入手机号和密码

**方式二：短信验证码登录**
- 输入手机号
- 获取短信验证码
- 输入验证码登录

**方式三：微信快捷登录**
- 点击「微信登录」
- 授权绑定
- 首次绑定需设置密码

## 2. 超级管理员功能

### 2.1 数据看板

登录后进入系统首页，查看：

- 租户总数 / 今日新增
- 活跃用户数
- 课程总数 / 学习总人次
- 系统收入统计（如有）

### 2.2 租户管理

#### 2.2.1 创建企业租户

1. 进入「租户管理」→「新增租户」
2. 填写企业信息：
   - 企业名称
   - 联系人姓名
   - 联系人手机
   - 套餐类型（基础版/标准版/企业版）
3. 设置套餐限制：
   - 学员上限
   - 课程数量上限
   - 月度导出次数
   - 存储空间限制
4. 设置到期时间
5. 点击「保存」

#### 2.2.2 租户套餐说明

| 套餐 | 学员上限 | 课程上限 | 导出次数 | 存储空间 | 价格 |
|------|----------|----------|----------|----------|------|
| 基础版 | 100 | 50 | 100/月 | 5GB | ¥999/年 |
| 标准版 | 500 | 200 | 500/月 | 20GB | ¥3999/年 |
| 企业版 | 2000 | 1000 | 2000/月 | 100GB | ¥12999/年 |

#### 2.2.3 租户续费

1. 找到对应租户
2. 点击「续费」
3. 选择续费时长（1 年/2 年/3 年）
4. 更新到期时间

#### 2.2.4 租户禁用

对于违规或欠费租户：
1. 点击「禁用」
2. 填写禁用原因
3. 确认后租户将无法登录

### 2.3 系统配置

#### 2.3.1 短信服务配置

1. 进入「系统配置」→「短信设置」
2. 选择服务商（阿里云/腾讯云）
3. 填写 AccessKey 和 Secret
4. 设置短信签名
5. 测试发送

#### 2.3.2 微信配置

1. 进入「系统配置」→「微信设置」
2. 填写小程序 AppID
3. 填写 AppSecret
4. 配置服务器域名（小程序后台）

#### 2.3.3 存储配置

1. 进入「系统配置」→「存储设置」
2. 选择存储方式（本地/对象存储）
3. 如使用 OSS/COS，填写对应配置

### 2.4 管理员管理

1. 进入「管理员管理」
2. 点击「新增管理员」
3. 填写用户名、密码、角色
4. 分配权限

## 3. 企业管理员功能

### 3.1 组织架构管理

#### 3.1.1 创建部门

1. 进入「组织架构」→「部门管理」
2. 点击「新增部门」
3. 填写部门名称、负责人
4. 设置上级部门（支持多级）
5. 点击「保存」

#### 3.1.2 批量导入员工

1. 下载导入模板
2. 填写员工信息（姓名、手机号、部门、岗位、工号）
3. 上传 Excel 文件
4. 系统自动发送注册通知短信

**模板格式**:

| 姓名 | 手机号 | 部门 | 岗位 | 工号 |
|------|--------|------|------|------|
| 张三 | 13800138000 | 技术部 | 工程师 | 001 |
| 李四 | 13900139000 | 销售部 | 经理 | 002 |

### 3.2 员工管理

#### 3.2.1 查看员工列表

- 按部门筛选
- 按认证状态筛选
- 搜索姓名/手机号

#### 3.2.2 员工详情

查看：
- 基本信息
- 实名认证状态
- 学习进度统计
- 学习记录明细

#### 3.2.3 禁用/启用员工

对于离职员工：
1. 找到对应员工
2. 点击「禁用」
3. 该员工将无法登录

### 3.3 课程管理

#### 3.3.1 创建课程

1. 进入「课程管理」→「新增课程」
2. 填写课程信息：
   - 课程名称
   - 课程分类（安全生产/法务合规/财税合规/行业专项）
   - 课程类型（视频/文档/混合）
   - 是否必修
3. 上传课程资源：
   - 封面图片（建议 750x400）
   - 视频文件（支持 mp4，最大 500MB）
   - 文档文件（支持 pdf/doc/xls，最大 50MB）
4. 填写课程介绍
5. 点击「保存」

#### 3.3.2 课程上架/下架

- 上架：员工可见并可学习
- 下架：员工不可见，已学习记录保留

#### 3.3.3 课程分类管理

1. 进入「课程管理」→「分类管理」
2. 新增/编辑/删除分类
3. 设置分类排序

### 3.4 学习任务

#### 3.4.1 创建学习任务

1. 进入「学习任务」→「新增任务」
2. 填写任务信息：
   - 任务名称
   - 选择课程（可多选）
   - 目标人员（全员/指定部门/指定岗位/指定人员）
   - 截止时间
   - 是否必修
3. 点击「发布」

#### 3.4.2 任务下发

任务发布后：
- 目标人员收到学习通知
- 课程出现在「必修课程」列表
- 系统开始统计进度

#### 3.4.3 任务监控

查看：
- 应学人数
- 已学人数
- 完成率
- 未完成人员名单

### 3.5 进度监控

#### 3.5.1 整体概览

查看企业整体学习数据：
- 总学习人次
- 总学习时长
- 课程完成率
- 认证率

#### 3.5.2 按部门统计

查看各部门学习情况排名：
- 部门完成率
- 部门平均学习时长
- 未完成人数

#### 3.5.3 未完成人员

查看逾期未完成学习的人员：
- 姓名、部门、岗位
- 应学课程
- 当前进度
- 逾期天数

### 3.6 报表导出

#### 3.6.1 导出个人报告

1. 进入「报表导出」→「个人报告」
2. 选择员工
3. 选择格式（Excel/PDF/Word）
4. 点击下载

**报告内容**:
- 员工基本信息
- 已学课程列表
- 每门课程的学习进度
- 学习时长统计
- 完成时间

#### 3.6.2 批量导出

1. 进入「报表导出」→「批量导出」
2. 选择导出范围：
   - 全员导出
   - 按部门导出
   - 按岗位导出
   - 按时间筛选
3. 选择格式（Excel/ZIP）
4. 点击「生成报告」
5. 下载文件

**ZIP 格式**: 包含每个员工的独立 PDF 报告

#### 3.6.3 导出记录

查看历史导出记录：
- 导出时间
- 导出人
- 文件类型
- 下载次数
- 过期时间（7 天）

### 3.7 操作日志

查看企业内所有操作记录：
- 登录日志
- 数据修改
- 报告导出
- 课程管理

## 4. 员工学习端

### 4.1 实名认证

首次使用需完成实名认证：

1. 进入「我的」→「实名认证」
2. 填写真实姓名、身份证号
3. 上传身份证正反面照片
4. 提交审核

审核通过后：
- 显示「已认证」标识
- 可生成正式学习报告

### 4.2 学习课程

#### 4.2.1 课程列表

- 必修课程（红色标识）
- 选修课程
- 按分类筛选

#### 4.2.2 视频学习

1. 点击课程进入详情
2. 点击「开始学习」
3. 视频播放器功能：
   - 播放/暂停
   - 进度拖拽（限制快进）
   - 倍速播放（0.5x - 2.0x）
   - 全屏切换
   - 后台播放（记录进度）

**防作弊提醒**:
- 请勿使用外挂刷课
- 请勿大跨度拖拽进度
- 系统会记录异常行为

#### 4.2.3 文档学习

1. 打开文档
2. 阅读内容
3. 系统自动记录阅读时长
4. 标记为「已读」

### 4.3 学习进度

#### 4.3.1 进度查看

进入「我的」→「学习记录」:
- 已学课程数
- 已完成课程数
- 总学习时长
- 每门课程进度

#### 4.3.2 继续学习

点击「继续学习」:
- 从上次离开的进度继续
- 断点续播

### 4.4 学习报告

#### 4.4.1 查看个人报告

进入「我的」→「学习报告」:
- 查看在线报告
- 下载 Excel/PDF 版本

#### 4.4.2 报告用途

- 合规培训证明
- 年度培训记录
- 职称评定材料
- 企业存档

## 5. 常见问题

### 5.1 员工无法登录

**问题**: 提示「用户不存在」或「密码错误」

**解决**:
1. 确认手机号是否正确
2. 联系企业管理员确认是否已导入
3. 使用短信验证码登录
4. 重置密码

### 5.2 视频无法播放

**问题**: 视频加载失败或卡顿

**解决**:
1. 检查网络连接
2. 切换 WiFi/4G 网络
3. 清理浏览器缓存
4. 更换浏览器尝试
5. 联系管理员确认视频文件是否正常

### 5.3 学习进度不更新

**问题**: 学完后进度仍显示 0%

**解决**:
1. 确保网络正常
2. 等待 1-2 分钟同步
3. 退出重新进入课程
4. 检查是否完整观看视频
5. 联系管理员查看后台记录

### 5.4 身份证上传失败

**问题**: 提示「图片过大」或「上传失败」

**解决**:
1. 确认图片大小（不超过 5MB）
2. 确认图片格式（jpg/png）
3. 重新拍照上传
4. 确保光线充足、字迹清晰

### 5.5 报告导出失败

**问题**: 点击导出无响应或报错

**解决**:
1. 检查是否有学习记录
2. 确认导出次数是否用完
3. 联系企业管理员
4. 稍后重试

### 5.6 租户到期

**问题**: 提示「租户已到期，请联系管理员」

**解决**:
1. 联系企业负责人
2. 联系系统超级管理员续费
3. 续费后可恢复正常使用
