# 发布说明 (Release Notes)

## [1.0.0] - 2026-04-19

### ✨ 新增功能

#### 核心功能
- **三重实名认证系统**
  - 姓名 + 手机号 + 身份证唯一绑定
  - 身份证照片上传与审核
  - AES-128-ECB 加密存储敏感数据
  
- **多租户 SaaS 架构**
  - 企业数据完全隔离
  - 租户套餐配置（基础版/标准版/企业版）
  - 租户到期管控
  
- **学习进度追踪**
  - 视频学习进度实时同步
  - 文档阅读记录
  - 断点续播功能
  - 心跳机制防止进度丢失
  
- **防作弊系统**
  - 大跨度拖拽检测
  - 频繁切页监控
  - 学习行为日志记录
  - 异常行为标记
  
- **报表导出模块**
  - Excel 个人/批量导出（PhpSpreadsheet）
  - PDF 报告生成（Dompdf）
  - Word 文档导出（PhpWord）
  - ZIP 批量打包下载

#### 认证模块
- 账号密码登录
- 短信验证码登录
- 微信快捷登录（框架）
- 密码 bcrypt 加密

#### 企业管理
- 组织架构管理（多级部门）
- 员工批量导入（Excel）
- 课程上传与分类
- 学习任务下发
- 进度监控看板

### 🏗️ 技术架构

#### 后端
- ThinkPHP 6.x 框架
- RESTful API 设计
- 中间件系统（认证/租户隔离/操作日志）
- MySQL 8.0+ 数据库
- 12 张核心数据表设计

#### 前端
- UniApp 3.x 多端框架
- H5 + 微信小程序 + APP 支持
- 组件化开发
- API 请求封装
